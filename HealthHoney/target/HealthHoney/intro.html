  1. 로그인(Google+)

보완 사항

운동 기록 측정(앱):
제공 서비스의 Google+ sns 글 공유(앱/웹):
기타 Google API 활용:
-현재는 타 앱이나 스마트폰 자체 수집한 자료를 출력하는데 주력
-Google API는 웹, 앱 접근 흐름이 같음.(OAUTH2.0)
=> 새로운 기능 추가에 대한 유연성.
=> 수집한 자료의 오차범위 축소.
=> 서버와의 동기화 시간 단축.


플로우


후기

김영준: 외부 API를 사용하는 것이라 쉽게 할 수 있을 것이라 생각했지만 상상이상으로 방대한 데이터를 다루는 것이 어려웠다. 대신 그만큼 많은 것을 배울 수 있어서 보람됬다.
김용현: 그저 만들어져 있는 API를 가져다 쓴다고 하여서 쉬울줄 알았는데 쓰는방법 또한 어려웠고 새로운 과제를 해결하는것이 재미 있었다. 영어공부는 필수다...
지승호: 새롭고 생소한 API들을 사용해 작업하는 것이 힘들 었지만 그 만큼 성취감도 컸고, API를 통해 코딩을 하는 것이 어떤 것인지 조금 알 것 같다.
황은미: 새로운 부분을 공부해가면서 해야되서 힘들었지만 그만큼 보람도 있었고, 코딩 능력도 중요하지만 영어 실력도 그에 못지않게 중요하다는 것을 절실히 깨달았습니다.
